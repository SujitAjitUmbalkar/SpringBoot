# Spring Boot Projects – Real World Applications

This repository contains **complete Spring Boot projects** that demonstrate how different concepts come together to build **real-world backend applications**.

Unlike concept-only examples, these projects focus on:

* Application flow
* Project structure
* REST API design
* End-to-end functionality

> This repository complements the **spring-boot-concepts** repo, which focuses on individual concepts.

---
---

## 📌 Purpose of This Repository

The main goals of this repository are:

* To showcase **practical implementation** of Spring Boot
* To demonstrate **CRUD operations in real applications**
* To show **project-level understanding**, not just annotations
* To help learners understand how concepts work together

## 🧠 Concepts Used in These Projects

* Spring Boot project structure
* REST Controllers (@RestController)
* CRUD operations (POST, GET, PUT, DELETE)
* Request handling (@RequestBody, @PathVariable)
* Service and Repository layers
* Dependency Injection (@Autowired)
* In-memory / database-based storage
* Maven build lifecycle

---

## 🛠 How to Run a Project

1. Clone the repository
2. Open a project in IDE (IntelliJ / Eclipse / STS)
3. Make sure Java and Maven are installed
4. Run the main application class
5. Test APIs using Postman or browser

Each project folder contains its **own README.md** with:

* Project description
* API endpoints
* Sample requests
